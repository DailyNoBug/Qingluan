# PortDefine 引脚定义

## 文档概述

本文档定义四轴无人机使用STM32f401ret6开发板与外设之间连接的GPIO引脚定义，硬件设计与软件设计均应遵循本文档的GPIO引脚定义

如果引脚定义存在冲突，请及时沟通并且修改硬件和软件的端口选择，并且修改本文档。
